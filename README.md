# 实验一：Python视觉开发环境搭建与图像基本读写

## 实验目的
1. 掌握OpenCV和NumPy的基本用法
2. 学习图像的基本读写、显示和转换操作
3. 熟悉图像处理的基本流程

## 实验环境
- Python 3.x
- OpenCV (cv2)
- NumPy
- Matplotlib

## 安装依赖
bash
pip install opencv-python numpy matplotlib
## 实验内容
完成以下6个任务：

### 任务1：使用OpenCV读取测试图片
使用`cv2.imread()`函数读取图片文件。

### 任务2：输出图像基本信息
在终端打印图像的：
- 尺寸（高度×宽度）
- 通道数
- 数据类型

### 任务3：显示原图
使用Matplotlib显示原始图像。

### 任务4：转换为灰度图
将彩色图像转换为灰度图并显示。

### 任务5：保存处理结果
将灰度图保存为新文件。

### 任务6：NumPy简单操作
1. 输出某个像素值（如图像中心点）
2. 裁剪图像左上角区域并保存

## 文件说明
- `experiment1.py`：实验源代码
- `dog.jpg`：原始测试图片
- `gray_image.jpg`：处理后保存的灰度图
- `cropped_region.jpg`：裁剪的图像区域
- `README.md`：本说明文件

## 运行方法
bash
python experiment1.py
## 实验结果
程序运行后会：
1. 在终端显示图像的基本信息
2. 弹出窗口显示四张图像：
   - 原始图像
   - 灰度图像
   - 裁剪区域
   - 标记裁剪位置的原始图像
3. 生成处理后的图像文件

## 注意事项
1. 如果没有测试图片，程序会自动创建一个示例图片
2. 在WSL中运行需要配置X Server显示GUI窗口
3. 图像显示窗口需要手动关闭才能继续程序执行
